# Web Designer Portfolio

A modern, responsive portfolio website for web designers built with HTML, CSS, and JavaScript.

## Features

- ✨ **Modern Design** - Clean and professional layout
- 📱 **Responsive** - Works perfectly on desktop, tablet, and mobile
- 🎨 **Beautiful UI** - Gradient backgrounds and smooth animations
- 🔄 **Smooth Scrolling** - Easy navigation between sections
- 📧 **Contact Form** - Get in touch functionality
- 🖼️ **Portfolio Showcase** - Display your best work
- ⚡ **Performance** - Lightweight and fast-loading

## Sections

1. **Navigation** - Sticky navbar with smooth scrolling
2. **Hero** - Eye-catching introduction section
3. **About** - Personal information and statistics
4. **Portfolio** - Showcase your projects with hover effects
5. **Skills** - Display your expertise
6. **Contact** - Contact form and information
7. **Footer** - Social media links

## Getting Started

### Option 1: Local Development

1. Clone the repository:
```bash
git clone https://github.com/jaykeemoreno/web-designer-portfolio.git
cd web-designer-portfolio
```

2. Open `index.html` in your web browser

### Option 2: GitHub Pages

1. Go to your repository settings
2. Under "Pages", select the main branch
3. Your site will be available at `https://yourusername.github.io/web-designer-portfolio`

## Customization

### Edit Personal Information
- Update the hero section text in `index.html`
- Change contact details in the contact section
- Update social media links in the footer

### Update Portfolio Projects
Replace placeholder images in the portfolio section:
```html
<img src="your-image-url" alt="Project Name">
```

### Customize Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --dark-color: #1a1a1a;
    --light-color: #f8f9fa;
    --text-color: #333;
}
```

### Update Skills
Add or modify skill cards in the skills section of `index.html`
